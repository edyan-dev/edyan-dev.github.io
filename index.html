Battle Crown

Gesture-based mobile fighting game with physics impact, where famous historycal warriors face each other for the symbol of martial superiority - the Battle Crown.

Mercenary

Outsource project. Deep midcore isometric stealth-action mobile game in an alternative medieval setting. The game contains, but is not limited to, stealth combat system, noises and field of view, light-shadow mechanics, body drag and hide, freerun mechanics.

Bananarang Island

Casual mobile aiming/timing game, where Player has to feed monkeys with bananas on a tropical island. Other island inhabitants are not into bananas, so Player should avoid bumping them with one.

Bumpy Road

Outsource project. Hyper-casual mobile game, where player has to change wheel size to get past road obstacles and get away from police.

Planet resources

Casual mobile game about resource gathering balance. Humanity has to gather resources to survive, but ship must leave planet orbit before the ecosystem depletes completely and planet explodes.

Round Maze

Hyper-casual mobile game made in one day, where player controls current circle ring instead of a ball. Gravity changes in different directions.
